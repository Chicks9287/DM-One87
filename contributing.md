# Contributing to Omega-Stack-Master

We appreciate contributions to make this the ultimate pentesting hub!

## Guidelines
- Follow ethical standards: No malicious code.
- Code Style: Bash - shellcheck compliant; Python - PEP8, Black formatted.
- Commit Messages: Use conventional commits (e.g., feat: add feature).
- Testing: Add tests for new features.

## Process
1. Fork and clone.
2. Branch: `git checkout -b feat/your-feature`
3. Develop and test: Run `pytest core/dmOne87/tests/`
4. PR: Open against main.

## Tools
- Linting: `shellcheck scripts/*.sh`
- Formatting: `black core/dmOne87/*.py`
